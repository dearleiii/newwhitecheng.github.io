Research Projects

I'm interested in computer vision, machine learning, and computer systems. The efficiency perspective of those fields is the common thread that interests me.
My recent focus lies in on-device machine learning.


Resource-constrained DNN Design

Mobile and edge devices are often characterized by stringent resource constraints, such as energy consumption and model size. Furthermore, depending on the application domain, multiply-accumulate operations (MACs) or latency may also need to be constrained.

Course Projects

Defend Adversarial Examples for Object Detection with Scales

We want to understand how image scale correlates to the robustness of the adversarial examples that are used to fool object detectors.

We find that though image scaling is able to defend adversarial example, it is still vulnerable to adaptive attack that uses expectation-over-transformation.

Course project for CMU 10-701

A Study of Efficiency in Distributed Deep Reinforcement Learning

We are interested in the impact of parameter aggregation algorithms on sample complexity and communication efficiency. We specifically focus on analyzing elastic averaging updates in the reinforcement learning setting. Compared to A3C, we find that a adapted version of elastic averaging has on-par sample complexity and wallclock training time while having 32x less communication costs compared to A3C for the LunarLander environment.

Course project for CMU 10-703

What's Left for CPU When It Comes to Deep Neural Networks?

We find that even though GPU is the key role for deep neural network inferencing, CPU's capability still play an important role due to data preprocessing. Also, in platforms where CPU and GPU have shared main memory, the different type of applications that run on CPU will have different impact on the performance of the DNN running on GPU.

Runner Up Award for CMU course 18-743
